Gnome Software

sudo rpm-ostree install ffmpegthumbnailer p7zip unrar webp-pixbuf-loader

Other Software

sudo rpm-ostree install bubblemail clash ffmpeg-libs fish hugo neofetch neovim proxychains-ng syncthing youtube-dl

Fonts
